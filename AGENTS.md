# Demo Farm Manager Agent Notes

This repo is a presentation/demo farming app, cloned from the JTG Farms app structure but cleaned for fictional demo use.

## Rules

- Keep demo data fictional.
- Do not copy private JTG/Heatwole/family/customer data into this repo.
- Do not add fallback Supabase credentials or private env files.
- Prefer mobile-first layouts and simple language for farm operators.
- When adding persistence later, use a separate demo/client Supabase project and document the env vars.

## Quality Gate

Before handoff, run:

```bash
npm run lint
npm run build
```
